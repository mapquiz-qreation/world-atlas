## Atlas of Syncro Eras ロードマップ

### 現状（フェーズ1 完了）
- [x] 年代スライダーで時代を切り替える地図表示
- [x] 地域ごとのカラーマーカー
- [x] パターンタグフィルター（交易・外圧・帝国・宗教・知の革命）
- [x] マーカータップで詳細・関連用語・パターンバッジを表示
- [x] パターン解説バナー（共テ視点の共通構造の説明）

### フェーズ2（次の開発）
- [ ] パターン認識クイズ「このイベントのパターンはどれ？」を選択式で出題
- [ ] 「この時代に同じパターンを持つ国を全て選べ」組み合わせ問題
- [ ] 時代データの拡充（現在5時代 → 20時代以上を目標）

### フェーズ3（将来）
- [ ] 並べ替えクイズ（出来事カードをスライダー順に並べる）
- [ ] 因果チェーンクイズ「この次に何が起きた？」選択肢形式
- [ ] Atlas of Historical Sites との連携（地図クイズ正解後に同時代情報を表示）
- [ ] ユーザースコア・復習モードをSyncro Erasにも対応

### コンセプトメモ
- Atlas of Historical Sites（場所を覚える）× Atlas of Syncro Eras（時代・パターンを覚える）
- 2本柱で共テ世界史の「地図問題・時系列・背景理解」をほぼカバー
- インプット（パターンを見て気づく）→ アウトプット（クイズで確認）の流れ
